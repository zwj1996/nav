# Nuxt 3 开发模板

集成了nuxt3 + vue3.4 + TypeScript + vueuse + Element-plus + Tailwind + EsLint + Prettier <br/> 1.对axios进行了二次封装 <br/>2.内置了一些常用工具函数 <br/> 3.内置了基础的登录页和两个简单的页面

## 下载依赖包

```bash
# pnpm
pnpm install
```

## 启动服务

启动后项目地址 `http://localhost:3000`:

```bash
# pnpm
pnpm run dev --host
```

## 项目打包

打包后项目文件夹为output

```bash
# pnpm
pnpm run build
```
